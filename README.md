# Chronic Disease Risk Modeling

NHANES-first research pipeline for chronic disease risk feature engineering, model training, evaluation, and a defense-facing web demo.

## Setup

1. `python -m pip install -r requirements.txt`
2. Run pipeline tests: `python -m pytest tests -q`

## Planned Workflow

1. `python scripts/download_nhanes.py`
2. `python scripts/build_nhanes_interim.py`
3. `python scripts/build_model_dataset.py`
4. `python scripts/train_baseline.py`
5. `python scripts/train_candidates.py`
6. `python scripts/export_artifacts.py`
7. `python scripts/run_nhanes_pipeline.py`

## Web Demo (答辩展示台)

The demo contains:
- Backend: FastAPI inference and artifact endpoints (`/api/*` + `/static/*`)
- Frontend: React + Vite single-page dashboard

Note: the demo reads evaluation artifacts from eports/tables/ (CSV + PNG) generated by the pipeline.

### Option A: One server (recommended for defense)

1. Build frontend
   - `cd frontend`
   - `npm install`
   - `npm run build`
2. Run backend
   - `cd ..`
   - `uvicorn backend.app.main:create_app --factory --reload`
3. Open: `http://127.0.0.1:8000/`

### Option B: Two servers (development)

1. Backend
   - `uvicorn backend.app.main:create_app --factory --reload`
2. Frontend
   - `cd frontend`
   - `npm install`
   - `npm run dev`
3. Open: `http://127.0.0.1:5173/`

## Outputs

- `reports/tables/`: metrics CSV output and plots used by the demo
- `reports/figures/`: plots and explanation figures
- `artifacts/`: web-facing JSON payloads
